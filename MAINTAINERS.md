## Maintainer responsibilities

Maintainers are responsible for:

- reviewing issues and pull requests
- keeping the shared theme, OG template, SEO contract, and banner pipeline consistent across the fleet
- enforcing the Code of Conduct
- triaging security reports through the private disclosure process
- protecting the design boundaries: dark-only, a single cyan signal accent, the eye as logo only, and the watercolor banner contract

## Project status

lidless-fleet-kit is early-stage open-source infrastructure for the Lidless Labs site fleet. The repository is public, MIT licensed, and actively maintained. The maintainer runs it against the real `lidless.dev` hub and treats outside issues and reproducible failures as the primary feature queue.

## Open-source program use

Credits, hosted services, or sponsored accounts granted to this project should be used only for the public open-source work: documentation, CI, release automation, dependency and security scanning, and reproducible maintainer workflows. They should not be used for unrelated private or closed commercial work.
